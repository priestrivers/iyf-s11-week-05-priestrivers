## Project Description

This project demonstrates JavaScript DOM manipulation and event handling concepts. It includes practical exercises on selecting and traversing DOM elements, modifying page content, handling user interactions, validating forms, and building an interactive To-Do List application. The goal was to learn how to create dynamic and interactive web pages using JavaScript.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Visual Studio Code
- Git
- GitHub

## Features

- DOM element selection and traversal
- Dynamic content and style manipulation
- Creating and removing HTML elements
- Event listeners for mouse, keyboard, and form events
- Form validation using JavaScript
- Interactive To-Do List with add, delete, complete, and filter functionality

## How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/priestrivers/iyf-s11-week-05-priestrivers.git
   ```

2. Navigate into the project folder.

3. Open the project in Visual Studio Code.

4. Launch `index.html` using **Live Server** or open it directly in your web browser.

## Lessons Learned

During this project, I learned how to:

- Select and manipulate HTML elements using the DOM.
- Traverse parent, child, and sibling elements.
- Modify text, HTML, styles, and attributes dynamically.
- Create, insert, clone, and remove DOM elements.
- Handle mouse, keyboard, and form events using event listeners.
- Apply event bubbling and event delegation.
- Validate user input before form submission.
- Build interactive web applications using JavaScript.
